Projeto Aplicado - Jogos digitais


Plataforma Brain Games

Este projeto é desenvolvido pelos alunos do 5° Semestre do Curso de Sistemas de Informação do Centro Universitário Una

A proposta do Projeto Aplicado é de desenvolver um projeto relacionado com a área de jogos.

Com base nesta proposta, o nosso Projeto visa uma plataforma de jogos online que venha gerar entretenimento e pararelamente desenvolver o raciocíno lógico dos jogadores.
Aliando assim a diversão com aprendizado.
